## How to use on your local machine

### Requirements:
* cmd (if on linux: terminal emulator & on mac: terminal)
* git (https://git-scm.com/downloads)
* Python 3.6 or newer

```
git clone https://github.com/sytespace/bot.git
cd bot
pip3 install 
```
